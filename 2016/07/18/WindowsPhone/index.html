WindowsPhone 刷机那些事儿

折腾手机，免不了要去刷机云云。作为流行的 Android 和 iOS 系统，其刷机教程冗长，光是装驱动这一块就能使一大部分人望而生畏了，整个流程下来都可以写一篇论文了，有些教程甚至比装 Windows 还要繁琐。而作为 1% 的 WindowsPhone 刷机真的太简单了，手机连上电脑点几下鼠标就刷完机了，比装 Windows 都要简单。

Step1 备份重要数据

打开 WindowsPhone ，进入设置—更新和安全—备份—高级选项
选择需要备份的数据，微软说备份的时间大概在 15 分钟左右，如果没有重(xiao)要(huang)数(pian)据(er)的话，备份这一步可以直接跳过。

Step2 安装刷机工具 —— Windows Device Recovery Tool

先来看看 Windows Device Recovery Tool 有啥用

你可以在电脑上下载 Windows Device Recovery Tool 并将其用于修复手机上的软件问题。它可以安装经批准可用于你的手机的最新版本的 Windows 和删除其他所有内容，包括应用、游戏、文本、呼叫历史记录、音乐和照片。在使用此工具之前，请查看重新启动或重置手机是否能修复该问题。

来源：Windows Device Recovery Tool： 常见问题

安装要求:

  1. 一根用于将手机连接到电脑的 USB 电缆。
  2. 运行 Windows 7、Windows 8、Windows 8.1 或 Windows 10 的电脑。
  3. 至少 4 GB 的电脑可用存储空间。
  4. 电脑能够正常访问微软服务器。

下载地址：http://go.microsoft.com/fwlink/p/?LinkId=522381

下载完成后双击运行，一路点下一步完成安装。

Step3 开始刷机

  1. 使用 USB 线连接手机，Windows Device Recovery Tool会自动识别手机
  2. 选择当前的手机，Windows Device Recovery Tool 开始联网查找固件
  3. 点击重新安装软件，接下来是一系列刷机前的准备，随后进入固件下载界面
  4. 下载完成后就进入了安装步骤
  5. 安装完成

Step4 配置手机

手机刷机完成后，需要进行一系列设置，比如登录微软账号云云，不过一开始是英文界面，但是你连 WindowsPhone 都敢用了，还会怕 English ？

好吧，我成功把 Windows 10 刷成了 Windows 8。

PS： 使用预览版系统

预览版系统对于更新狂魔来说再爽不过了，但如何能收到推送呢？

  1. 加入 Windows Insider，可以在这里提交申请
  2. 进入应用商店，搜索Windows Insider
  3. 安装完成后打开 Windows Insider
  4. 选择升级通道
通道 功能 风险 更新频率
Insider Fast 第一时间获得系统新版本和功能更新 较高 快
Insider Slow 获得早期系统版本和功能更新 一般 一般
Insider Release Preview 更早获得 Current Branch 更新、微软应用程序和驱动 低 慢

  1. 选择完成后设备会重启，重启后进入手机更新，此时便会收到推送

  2. 更新完成

相较于 Android 和 iOS系统 WindowsPhone 刷机教程真的是简洁明了，不像外面那些妖艳的刷机教程。这也正是 WindowsPhone 的优点之一。

1% 代表的不是辣鸡，更多的是一种小众的优质。
