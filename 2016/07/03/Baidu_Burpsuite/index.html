使用 Burpsuite 自定义百度云提取码

之前写了一篇《使用 Fiddler 自定义百度云分享提取码》，简书有朋友@代码描绘人生提示说Burpsuite也可以实现该功能，遂决定再写一篇 Burp 的教程。

Step1 下载与安装 Burp

1.Burp 基于 Java，需要先安装 Java 运行环境（Java Runtime Environment）

系统 下载地址
Win x86 Oracle官网下载
Win x64 Oracle官网下载

2.下载 Burp

访问下载界面，Burp 有免费版，也提供功能更为全面的专业版。


下载完成后双击 jar 文件运行即可启动 Burp

Step 2 修改代理设置

1.打开 Burp
2.进入Proxy/Options
3.勾选运行代理（默认状态为勾选，如果项目为空，请按下图自行添加）

3.打开IE浏览器中的Internet选项
4.切换到连接标签
5.点击局域网设置
6.勾选为 LAN 使用代理服务器
7.在地址项目中填入 127.0.0.1，在端口项目中填入 8080 保存即可。

Burp 使用完后请取消勾选第6步中的为 LAN 使用代理服务器，否则将无法上网

如果使用Chrome浏览器，可以安装 Proxy SwitchyOmega 扩展来快速添加浏览器代理，添加方法不再赘述。


代理设置完成后，浏览器中的数据包都会通过 Burp 这个中转站。

Step3 修改提取码

1.首先切换到 Burp 界面，进入Prxoy/Intercept
点击Intercept Is On按钮关闭 Intercept 功能

2.打开配置好代理的浏览器，访问百度云
3.选中需要分享的文件，点击分享
4.点分享按钮出现如下界面


5.切换到 Burp 界面，开启 Intercept 功能
6.切换到浏览器，点击创建私密连接
7.此时本地生成的提取码被 Burp 阻止，Burp 会有提示
8.切换到 Burp 界面，即可看到被阻止的包，末尾的

pwd=s84y

即为本地生成的提取码元数据

9.删掉原来的提取码，直接修改 pwd 后面的值即可
Burp 对中文不友好，目前只能将提取码修改为数字或英文，并不能像 Fiddler 一样在提取码中添加中文


10.修改完成后，点击Intercept Is On按钮关闭 Intercept 功能，将包放行，此时带着修改提取码的包便会发送给服务器，也就实现了提取码修改。

总结

Burp 和 Fiddler都有各自好处，两者都是抓包和重放的利器，两者功能也远不止修改提取码这么简单，都值得深入学习。
