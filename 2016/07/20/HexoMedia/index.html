hexo-tag-dplayer:https://github.com/NextMoe/hexo-tag-dplayer

两款插件基于 DIYgod 编写的 html5 播放器 APlayer 和 DPlayer 开发。

APlayer 与 DPlayer 均支持https。

项目 使用难度 是否支持https 是否使用 html5
插入 html 语言 小 大部分不支持 大部分使用 Flash
使用 Hexo 插件 中等 支持 使用 html5

上述两种方法各有优缺点，选择一种适合自己的方法，打造属于自己的个性 Hexo 博客吧。
