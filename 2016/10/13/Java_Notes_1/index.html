Java 学习手记 —— 使用 JDK 编写第一个 Java 程序


大一学习的是计算机网络专业，要求学习 Java 语言，以后博客会开设一个专题，做一点学习 Java 的笔记。
JDK：Java Development ToolKit，Java 开发工具包，学习 Java 必装。

编写 Java 程序，实际上是对 .java 文件的编写，文件编写完成后使用编译器编译为二进制 .class 文件，再在 Java 虚拟机（Java Virtual Machine，简称 JVM）中运行 Java 程序。
Java 特点为：一次编译，到处执行。

Step1 JDK 的安装:

首先在甲骨文网站上下载JDK：http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
下载完成后执行安装即可

Step2 JDK 的环境配置

使用 JDK 必须配置 Path 环境变量，以及 classpath 系统变量。
1.右击此电脑，选择属性
2.左侧选择高级系统设置
3.进入环境变量
4.在系统变量重找到Path项目，点击编辑
5.将 JDK 目录添加进Path中，默认添加路径为


6.保存后返回系统变量设置界面
7.新建一个系统变量，变量名为classpath，变量值为任意目录，比如D:\WorkSpace

（为方便后期操作，建议所有操作都在D:\WorkSpace目录中进行）
