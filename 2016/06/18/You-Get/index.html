You-Get——基于Python3的媒体下载工具

You-Get是一个基于 Python 3 的下载工具。使用 You-Get 可以很轻松的下载到网络上的视频、图片及音乐。目前 You-Get 支持的站点达到了 71 个，囊括了国外的 Youtube、Vimeo、Tumblr、Instagram 等以及国内的爱奇艺、优酷、乐视、哔哩哔哩等。

项目主页：https://github.com/soimort/you-get
You-Get主页：https://you-get.org/
作者：soimort

Python 是一款跨平台编程语言，Liunx 以及 Mac OS 下都可以使用，此处展示仅使用 Windows。

Step1 下载与安装 Python 3

1.访问 Python 官方网站下载对应版本的Python，You-Get基于Python 3，建议下载 Python 3.5.1。
2.安装 Python 3.5.1，注意勾选Add Python 3.5 to PATH

3.按Win+R键打开运行，输入cmd，再输入python并回车执行，如果出现如下界面，则代表安装成功。

0x02 You-Get 有关视频的其他用法

0x03 You-Get 下载图片

You-Get 不但支持视频下载，同样支持图片下载，支持的站点已在其主页列出，You-Get 图片下载方式按站点大概分为如下 3 种，请对应站点选择合适方式。

1.需要进入图片/帖子详情页下载

适用于Twitter、Pinterest、Imgur、Flickr 等
1.访问网站
2.点击图片/帖子，进入图片/帖子详情
3.复制地址到 You-Get 下载即可

截图中展现了一些例子

2.支持列表/专辑下载

适用于花瓣、百度贴吧、Tumblr 等
1.访问网站，进入列表/专辑
2.复制地址到 You-Get 下载即可
（百度贴吧可以下载某个帖子中的全部图片，如果帖子中有优酷视频，则视频也会一并下载，如果视频为贴吧视频，则不会下载）
