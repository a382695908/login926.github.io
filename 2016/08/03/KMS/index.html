大量授权密钥（英语：Volume License Key，简称VLK）是软件公司使用的一个词汇，用户购买批量授权，便会获得一产品密钥，该密钥可安装于多部电脑。
来源：维基百科：大量授权密钥

所以我们可以通过自建 KMS 服务器来实现 Windows 以及 Microsoft 软件激活。相比于流行的本地 KMS 软件，在路由器上建立 KMS 服务器可以更易于管理，也不必担心激活时被植入恶意软件或修改主页。

1.准备好路由器并刷入 OpenWrt 系统

Newifi 这款路由器支持刷入 OpenWrt 系统
如果不清楚自己的路由器是否能够刷 OpenWrt 系统，请尝试在搜索引擎中输入

路由器型号 Openwrt

关键字进行搜索。
如果路由器支持刷入 OpenWrt 系统，我建议去刷集成 KMS 服务的固件。
如果找到不到 集成 KMS 的固件，可以参考这篇文章通过 opkg 安装 KMS 服务。

Newifi 刷机教程可以参考 Newifi 论坛，建议刷入这款固件集成了 KMS 服务。（感谢贴吧网友幼幼保护协会在这个帖子中的分享）

2. 进入路由器后台开启 KMS 服务

进入服务-KMS 服务器，勾选开启

Microsoft 在今天推送了周年更新，更新的名称会是 Windows 10 功能更新 Version 1607，版本号是 Build 14393.10。
这次更新带来了原生基于 Ubuntu Linux 的命令行、Edge 浏览器改进、Cortana 优化、Ink 墨迹书写/屏幕涂鸦、面部识别登录、XBox 游戏跨平台支持等新功能。

不过使用下来也发现了一些小问题：

  1. 迅雷极速版频繁崩溃（更新至 v1.0.34.360 版本可以解决）
  2. MacType 无法自启动
  3. 个性化中的颜色竟然没有草绿色
  4. Cortana 增加了查找手机功能，但是整个过程时间太长
